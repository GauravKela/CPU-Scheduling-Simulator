# 💻 CPU Scheduling Simulator

## 📌 Overview

This project is a GUI-based CPU Scheduling Simulator built using Python and Tkinter. It demonstrates how different CPU scheduling algorithms work in Operating Systems.

---

## 🚀 Features

* ✅ First Come First Serve (FCFS)
* ✅ Shortest Job First (SJF)
* ✅ Round Robin (RR)
* ✅ Gantt Chart Visualization
* ✅ Interactive GUI

---

## 🧾 Input

* Number of processes
* Arrival Time (AT)
* Burst Time (BT)
* Time Quantum (for Round Robin)

---

## 📊 Output

* Waiting Time (WT)
* Turnaround Time (TAT)
* Visual Gantt Chart
